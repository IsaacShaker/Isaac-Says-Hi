Git Training

I designed an interactive Git training with Figma (UI/UX design tool). The main purpose of this project was to teach others how to use Git. This was accomplished by helping them visualize commands in a mock terminal. Try Here

I utilized Figma components, variations, and protoyping to create an informative, user-friendly, and interactive platform. Users first learn about what Git is and why it is used. Then, they learn the basic Git commands. Finally, their knowledge is accessed with a multiple choice quiz.

Assembly Elevator

The purpose of this project is to mimic the functionality of an elevator using Assembly Language and an STM32-Nucleo microcontroller. The elevator has 4 floors and can move up and down. The user can press a button to call the elevator to their floor. The elevator motors will then move to the desired floor and open its doors.

Our team of 3 created this project for our Embedded Systems class. A full design document can be found on our Github repository.

DLA Simulation

This is an interactive website that helps users visulaize, and model, diffusion limited aggregation (DLA). The simulation is centralized around a middle particle(s) known as the aggrogate. Particles start at the perimeter of the scene and randomly move until a collision with the aggrogate is made. When a particle touches the aggrogate, it sticks and inherits its color. This forms a fractal like shape that resembles a snowflake. Try Here | Github Link
